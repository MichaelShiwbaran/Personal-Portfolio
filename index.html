Here are some of my projects I've been working on

  • Social Media Project

    C++

    This project is meant to be the backend of a social media website where users can create and account username and password. There is an account system implemented where users cannot make accounts with the same usernames as well.

  • Login/Registration Project

    C++

    This project allows users to create an account that has a username and password. These piece of information are stored in a file made when the user first registered. The user can then log in so long as the username and password are correct by comparing their login credentials with what is on the created file.

  • Rock Paper Scissors Project

    HTML, CSS, JavaScript

    This project is a website where users can go to learn about the game Rock, Paper, Scissors. There is filler information about two people who want to teach users how to play, and there is an actual Rock, Paper, Scissors game that the user can play against the computer with predetermined outcomes, such as rock beats scissors, scissors beats paper, and paper beats rock.

  • High-Low Number Project

    HTML, CSS, JavaScript

    This project is a website where users can play a game where they guess a number between 1 and 20. The number is randomly generated when the webpage is first launched and will tell the user if they are too high, or too low when they guess. When the correct number is guessed, the correct number changes instantly so the user can continue to play if they choose.

  • Advice Project

    HTML, CSS, JavaScript

    This project is a website where users can go to get a piece of advice if they need it. This can range from compliments to advice on how to go about your day. The pieces of advice are collected and displayed using an api. The user can also continue to generate more pieces of advice by clicking the more advice button.
